## Тестовые сценарии
| N | Действие | Описание и результат |
|-|-|-|
||**Любой пользователь**||
| 1 | Валидная авторизация | Открывается личный кабинет пользователя |
| 2 | Невалидная авторизация | Вывод сообщения об ошибке;<br>Остается форма авторизации (не происходит переход в личный кабинет) |
| 3 | Просмотр расписания группы или преподавателя | Для неавторизованных пользователей выводится список филиалов, для каждого из которых можно посмотреть расписание отдельной группы или преподавателя. Авторизованным пользователем выводится расписание относящиеся к их профилю, например, студенту - расписание его группы, при этом можно посмотреть расписание во всех филиалах. |
| 4 | Выход из аккаунта (только для авторизованных) | Открывается главная страница, и пользователь значится как "неавторизованный" |
| 5 | Изменение личных данных для входа в систему (только для авторизованных) | Открывается форма добавления/изменения личных данных пользователя и сохранение изменений в системе |
| 6 | Просмотр списка специальностей по которым производится подготовка в вузе | Вывод списка всех специальностей и их краткое описание |
||**Администратор системы**||
| 1 | Валидное добавление нового пользователя | Сохранение информации о новом пользователе в системе |
| 2 | Невалидное добавление нового пользователя | Вывод ошибки о некорректных или неполных данных |
| 3 | Валидное редактирование учетной записи пользователя | Выбор определенного пользователя;<br>Редактирование информации о пользователе;<br>Сохранение изменений в системе |
| 4 | Невалидное редактирование учетной записи пользователя | Вывод ошибки о некорректных или неполных данных
| 5 | Удаление пользователя | Выбор существующего пользователя; <br> Удаление пользователя из системы |
||**Руководство института**||
| 1 | Валидное добавление/изменение нового филиала | Создание и добавление информации о новом филиале в системе/Редактирование информации о филиале;<br>Сохранение изменений в системе |
| 2 | Невалидные добавление/изменение филиала | Вывод ошибки о некорректных или неполных данных |
| 3 | Валидное добавление/изменение новой специальности | Создание и добавление информации о новой специальности в системе/Редактирование информации о специальности;<br>Сохранение изменений в системе |
| 4 | Невалидные добавление/изменение специальности | Вывод ошибки о некорректных или неполных данных |
| 5 | Валидное добавление/изменение кафедры | Создание и добавление информации о новой кафедре и ее специализации в системе/Редактирование информации о кафедре;<br>Сохранение изменений в системе |
| 6 | Невалидное добавление/изменение кафедры | Вывод ошибки о некорректных или неполных данных |
| 7 | Добавление учебного плана для специальности | Выбор специальности;<br>Добавление учебного плана;<br>Сохранение изменений в системе |
||**Руководство филиала**||
| 1 | Валидное добавление/изменение нового факультета | Создание и добавление/изменение информации о новом факультете;<br>Сохранение информации в системе |
| 2 | Невалидные добавление/изменение факультета | Вывод ошибки о некорректных или неполных данных |
| 3 | Просмотр расписания занятий | По умолчанию открывается расписание для данного филиала, есть возможность просмотра расписания других филиалов |
||**Сотрудники деканата**||
| 1 | Перевод студента в другую учебную группу (смена специальности/кафедры) | Удаление информации о студенте из старой учебной группы;<br>Добавление информации о студенте в новую учебную группу;<br>Сохранение изменений в системе |
| 2 | Перевод студентов на следующий курс | После ввода информации о приказе студенты из подготовленного на основе успеваемости списка переводятся на следующий курс |
| 3 | Отчисление студентов | Создание списка студентов на отчисление;<br>После ввода информации о приказе в профиле студентов появляется пометка «Отчислен» и номер соответствующего приказа;<br>Студент удаляется из списка группы в системе |
| 4 | Просмотр успеваемости студента по всем предметам | Выбор определенной кафедры;<br>Выбор определенной учебной группы;<br>Выбор определенного студента;<br>Вывод списка предметов и оценок(баллов) по промежуточному контролю и за сессию |
| 5 | Назначение стипендий | Назначение стипендии всем студентам из списка, подготовленного на основании успеваемости |
| 6 | Просмотр расписания занятий | По умолчанию открывается расписание данного филиала для данного факультета |
||**Сотрудники кафедры, организующие учебный процесс**||
| 1 | Определение преподавателей для проведения занятий в учебных группах по конкретному предмету | Вывод учебных групп;<br>Выбор учебной группы;<br>Вывод предметов данной учебной группы;<br>Выбор предмета данной учебной группы;<br>Выбор преподавателей;<br>Сохранение изменений в системе |
| 2 | Добавление/редактирование информации о научной деятельности кафедры | Добавление/изменение информации о научной деятельности кафедры;<br>Сохранение изменений в системе |
| 3 | Просмотр учебного плана | Вывод информации об учебном плане для выбранной специальности из специальностей, по которым проводится обучение на данной кафедре|
| 4 | Учет учебного оборудования  | Добавление нового оборудования в систему или изменение информации о состоянии оборудования |
| 5 | Просмотр расписания занятий | По умолчанию открывается расписание данного филиала для данного факультета для данной кафедры |
||**Преподаватель**||
| 1 | Просмотр учебного плана | Вывод информации об учебном плане для выбранной специальности из специальностей, по которым проводится обучение на данной кафедре|
| 2 | Добавление\изменение программы занятий по своим предметам| Добавление\изменение программы занятий по своим предметам;<br>Сохранение изменений в системе |
| 3 | Проставление оценок студента по своим предметам и своей учебной группе в соответствии с учебным планом | Выбор определенной учебной группы;<br>Выбор определенного студента в группе;<br>Добавление оценок(баллов) за определенное занятие/зачет/экзамен;<br>Сохранение изменений в системе |
| 4 | Просмотр успеваемости студентов по своим предметам | Выбор определенной учебной группы;<br>Выбор определенного предмета;<br>Вывод списка студентов и оценок(баллов) по промежуточному контролю и за сессию |
| 5 | Просмотр информации о научной деятельности | Вывод научных работ, к которым данный преподаватель имеет отношение |
| 6 | Добавление/редактирование информации о научной деятельности | Добавление/изменение информации об исследовании, к которому преподаватель имеет отношение|
| 7 | Просмотр расписания занятий | По умолчанию открывается расписание данного филиала для данного факультета для данной кафедры для данного преподавателя|
||**Студент**||
| 1 | Просмотр своей успеваемости | Вывод успеваемости студента по всем предметам (в данном семестре) с возможностью просмотра успеваемости за время всего обучения |
| 2 | Просмотр информации о научной деятельности | Вывод научных работ, к которым данный студент имеет отношение |
| 3 | Выбор предмета в начале семестра (если выбор  предусмотрен учебным планом) | Вывод информации о выборе предмета и возможные варианты;<br>Выбор предмета;<br>Сохранение информации о выбранном предмете в системе |
| 4 | Просмотр расписания занятий | По умолчанию открывается расписание данного филиала для данного факультета для данной кафедры для группы студента |
