## Тестовые сценарии
| N | Действие | Описание и результат |
|-|-|-|
||**Любой пользователь**||
| 1 | Валидная авторизация | Открывается личный кабинет пользователя |
| 2 | Невалидная авторизация | Вывод сообщения об ошибке;<br>Остается форма авторизации (не происходит переход в личный кабинет) |
| 3 | Просмотр расписания группы или преподавателя | Для неавторизованных пользователей выводится список филиалов, для каждого из которых можно посмотреть расписание отдельной группы или преподавателя. Авторизованным пользователем выводится расписание относящиеся к их профилю, например студенту - расписание его группы, при этом можно посмотреть расписание во всех филиалах. |
| 4 | Выход из аккаунта (только для авторизованных) | Открывается главная страница и пользователь значится как "неавторизованный" |
| 5 | Изменение личных данных для входа в систему (только для авторизованных) | Открывается форма добавления/изменения личных данных пользователя и сохранение изменений в системе |
||**Администратор системы**||
| 1 | Валидное добавление нового пользователя | Сохранение информации о новом пользователе в системе |
| 2 | Невалидное добавление нового пользователя | Вывод ошибки о некорректных или неполных данных |
| 3 | Валидное редактирование учетной записи пользователя | Выбор определенного пользователя;<br>Редактирование информации о пользователе;<br>Сохранение изменений в системе" |
| 4 | Невалидное редактирование учетной записи пользователя | Вывод ошибки о некорректных или неполных данных
| 5 | Удаление пользователя | Выбор существующего пользователя; Удаление пользователя из системы |
||**Руководство института**||
| 1 | Валидное добавление/изменение нового филиала | Создание и добавление информации о новом филиале в системе/Редактирование информации о филиале;<br>Сохранение изменений в системе |
| 2 | Невалидные добавление/изменение филиала | Вывод ошибки о некорректных или неполных данных |
||**Руководство филиала**||
| Руководство филиала | Валидное добавление/изменение нового факультета | Создание и добавление/изменение информации о новом факультете;<br>Сохранение информации в системе |
| 1 | Невалидные добавление/изменение факультета | Вывод ошибки о некорректных или неполных данных |
| 2 | Валидное добавление кафедры на факультет | Выбор определенного факультета;<br>Создание и добавление информации о новой кафедре;<br>Сохранение изменений в базе данных |
| 3 | Невалидное добавление кафедры на факультет | Вывод ошибки о некорректных или неполных данных |
| 4 | Просмотр расписания занятий | По умолчанию открывается расписание для данного филиала, есть возможность просмотра расписания других филиалов |
||**Сотрудники деканата**||
| 1 | Добавление учебных групп | Выбор определенной кафедры;<br>Добавление новой учебной группы;<br>Сохранение изменений в системе |
| 2 | Добавление (зачисление) студентов в учебные группы | Выбор определенной учебной группы;<br>Добавление студентов в учебную группу;<br>Сохранение изменений в системе |
| 3 | Добавление учебного плана для кафедры факультета | Выбор определенного факультета;<br>Выбор определенной кафедры; Добавление учебного плана;<br>Сохранение изменений в системе |
| 4 | Перевод студента в другую учебную группу | Удаление информации о студенте из старой учебной группы;<br>Добавление информации о студенте в новую учебную группу;<br>Сохранение изменений в системе |
| 5 | Отчисление студента | В профиле студента появляется пометка «Отчислен» и номер соответствующего приказа;<br>Студент удаляется из списка группы в системе |
| 6 | Просмотр успеваемости студента по всем предметам | Выбор определенной кафедры;<br>Выбор определенной учебной группы;<br>Выбор определенного студента;<br>Вывод списка предметов и оценок(баллов) по промежуточному контролю и за сессию |
| 7 | Просмотр расписания занятий | По умолчанию открывается расписание данного филиала для данного факультета |
||**Сотрудники кафедры, организующие учебный процесс**||
| 1 | Добавление/изменение расписания для группы кафедры | Добавление/изменение записи о расписании учебной группы;<br>Сохранение изменений в системе |
| 2 | Добавление/редактирование информации о научной деятельности кафедры | Добавление/изменение информации о научной деятельности кафедры;<br>Сохранение изменений в системе |
| 3 | Просмотр учебного плана | Вывод информации об учебном плане для выбранной группы |
| 4 | Просмотр расписания занятий | По умолчанию открывается расписание данного филиала для данного факультета для данной кафедры |
||**Преподаватель**||
| 1 | Просмотр учебного плана | Вывод информации об учебном плане для выбранной группы |
| 2 | Добавление\изменение плана занятий по своим предметам | Добавление\изменение плана занятий по своим предметам;<br>Сохранение изменений в системе |
| 3 | Проставление оценок студента по своим предметам | Выбор определенной учебной группы;<br>Выбор определенного студента в группе;<br>Добавление оценок(баллов) за определенное занятие/зачет/ экзамен;<br>Сохранение изменений в системе |
| 4 | Просмотр успеваемости студентов по своим предметам | Выбор определенной учебной группы;<br>Выбор определенного студента;<br>Вывод списка предметов и оценок(баллов) по промежуточному контролю и за сессию |
| 5 | Просмотр информации о научной деятельности | Вывод научных работ, к которым данный преподаватель имеет отношение |
| 6 | Просмотр расписания занятий | По умолчанию открывается расписание данного филиала для данного факультета для данной кафедры для данного преподавателя|
||**Студент**||
| 1 | Просмотр своей успеваемости | Вывод успеваемости студента по всем предметам (в данном семестре) с возможностью просмотра успеваемости за время всего обучения |
| 2 | Просмотр информации о научной деятельности | Вывод научных работ, к которым данный студент имеет отношение |
| 3 | Выбор предмета в начале семестра (если выбор  предусмотрен учебным планом) | Вывод информации о выборе предмета и возможные варианты;<br>Выбор предмета;<br>Сохранение информации о выбранном предмете в системе |
| 4 | Просмотр расписания занятий | По умолчанию открывается расписание данного филиала для данного факультета для данной кафедры для группы студента |
